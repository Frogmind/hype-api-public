# Host
## Summary
Host player helpers (position, focus) for the active session.

## API Reference

## getPosition

Returns the host player's world position if available.

### Signature

```luau
hype.host.getPosition(): vector|nil
```

## getFocusPosition

Returns the point on the ground the host is focusing on (insert location).

### Signature

```luau
hype.host.getFocusPosition(): vector|nil
```
